请升级您的浏览器，以便我们更好地为您提供服务

检测到您正在使用较旧的浏览器（例如 IE 内核或 IE11 以下）。该类浏览器缺少必要的安全更新与现代网页能力， 可能导致页面无法打开、功能异常或存在安全风险。

建议使用最新版 Microsoft Edge / Google Chrome / Mozilla Firefox 访问。

推荐浏览器

如果您的电脑已安装以下浏览器，请直接使用其最新版本访问此网站：

Google Chrome 速度快（常用） → Microsoft Edge 兼容性强 → Mozilla Firefox 隐私友好 →

注：如果您使用的是企业内网/旧系统自带的 IE 内核浏览器，请联系 IT 协助升级或切换到 Edge 的 IE 模式（仅在必须兼容旧系统时使用）。

为什么需要升级？

  • 旧浏览器缺少安全更新，容易被恶意软件攻击。
  • 不支持现代 Web 标准，可能导致页面白屏、按钮失效、无法登录等问题。
  • 兼容旧浏览器需要额外成本，且体验通常更差。
若升级后仍无法访问，请截图并联系管理员。
